# Write PAPER Content

`SKILL.md`의 실질적인 지침이 바뀌면 이 문서도 함께 갱신합니다.

## 중심

- 본문 초반에 공식 논문 링크를 넣습니다.
- 논문이 던지는 질문, 방법, 결과, 왜 중요한지를 중심에 둡니다.
- 논문 고유가 아닌 재사용 가능한 수학은 `THEORY`로 링크합니다.
- 코드 독해나 구현 실습은 `REPO` 또는 `IMPLEMENT`로 넘깁니다.

## 권장 구조

- 공식 소스
- 문제 설정
- 핵심 아이디어 또는 알고리즘
- 이전 작업과 달라진 점
- 왜 여기서 중요한가
- 로컬 관련 문서 링크

## 근거 규칙

- 실제 논문 PDF와 공식 arXiv 또는 출판사 페이지를 우선합니다.
- 논문이 직접 말하는 내용과 내 해석을 구분합니다.
- 논문 인용 때문에 `tracked_versions`를 쓰지 않습니다.
