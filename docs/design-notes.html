Design Notes

Seraph Field는 수학 지식을 노드-엣지 그래프로 탐색하는 인터페이스다. 이름은 헤븐 번즈 레드의 Seraph, 필드는 QFT(Quantum Field Theory)에서 따왔다. 시각 언어는 Psycho-Pass의 시빌라시스템·도미네이터 HUD를 참조하되, 다크 시안 배경에 녹아드는 tonal 처리를 원칙으로 한다.

Palette

전체 팔레트는 cyan-family 내 hue shift로 구성된다. 채도는 낮추고 명도 차이로 카테고리를 구분한다 — 배경 다크 시안에서 튀지 않으면서 읽힌다.

Node

노드는 어둡고 반투명한 구체다. 배경이 비쳐 보이고, 외곽에는 카테고리 색의 얇은 atmospheric rim glow가 감돈다 — 우주에서 본 행성의 대기층. 내부에 심볼·tick·하이라이트를 넣지 않는다.

Spatial Depth

깊이감은 배경 레이어링으로 만든다. 그래프는 이 공간 안에 떠 있는 레이어로 읽힌다.

  • Floor grid — 2D 평면 격자 (80px), 중앙에서 가장자리로 페이드아웃
  • Atmosphere — 좌우 vignette pool + 상단 fog + 하단 암흑
  • Ambient particles — 천천히 drift하는 quantum 입자 + 주기적 pulse ring

HUD Frame

  • 뷰포트 4코너에 시안 L자 브래킷 (22px) + drop-shadow 글로우

Sidebar

사이드바는 모바일 디바이스 셸 형태다. 우측 엣지에 베젤 그라디언트·전원 버튼·스피커 슬릿, 내부는 프로스티드 글래스 화면 (14px 라운드 코너).

  • 탭: FILTER · SEARCH · STATS
  • Edge type 셀은 코너-컷 inventory 그리드 스타일
  • 900px 이하에서 drawer로 전환

Interaction

  • 드래그·줌·필터·검색 — Cytoscape + fcose 레이아웃
  • 노드 클릭 → HUD 팝업 다이얼로그 (4코너 프레임, 카테고리 뱃지, 액션 버튼)
  • Ambient drift — 노드 위치에 slow noise 오프셋 적용 (토글 가능)
  • 100개 미만 최적화, 500+ 까지 성능 유지되는 구조
