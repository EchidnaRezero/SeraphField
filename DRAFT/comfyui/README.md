# ComfyUI Archive

이 폴더는 ComfyUI 분석 초안 보관용이다.

여기 있는 문서들은 사이트 공개용 원본이 아니다. 공개용 기준 문서는 `RAW/REPO/ComfyUI SDXL and Anima Sampling Paths.md`와 관련 `RAW/THEORY` 문서들이다.

## 포함 내용

- startup과 prompt 실행 흐름 초안
- SDXL와 Anima 로딩 및 샘플링 비교 초안
- SDXL U-Net과 Anima DiT 구조 초안
- 샘플링 수학 초안과 ver2 초안

## 사용 원칙

- 여기 있는 초안은 비교나 참고를 위해 남겨 둔다.
- 사이트용 내용을 다시 고칠 때는 먼저 `RAW/` 문서를 수정한다.
- 같은 내용을 `docs/`와 `RAW/`에 동시에 유지하지 않는다.
